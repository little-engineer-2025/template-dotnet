Common template

This repository contains common contents that repeat from one repo to other,
to make easier to create new templates faster and being consistent between them.

Common features:

- Contributing
- Security
- Automation to check:
  - Enforce conventional commits.
  - Auto releases and tagging based on conventional commits.
  - Automatically generate changelog.
- Base templates.
- Integration with **adr** tools.
- Common Makefile rules missing the specific general rules for your template.
